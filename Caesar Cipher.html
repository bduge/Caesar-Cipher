Bill's Caesar Cipher Generator

A caesar cipher is one of the oldest and simplest encryption technique that still remains fun to play with. It moves every letter in a given "secret message" down or up the alphabet by a given "shift amount". For example, the if the code was the message "abc" and the shift was 3, each letter in "abc" would be shifted down the alphabet 3 places to return the cipher "def". Make your own cipher below!
